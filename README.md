# Wadling

[![Gem Version](https://badge.fury.io/rb/wadling.png)](https://badge.fury.io/rb/wadling)

## Installation

Add this line to your application's Gemfile:

    gem 'wadling'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install wadling
